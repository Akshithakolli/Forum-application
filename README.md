# Forum Application

A web-based forum application that allows users to create posts, comment, and engage in discussions.

## Features
- User authentication
- Create, edit, and delete posts
- Commenting system
- Responsive UI

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Java / Spring Boot (edit if needed)
- Database: MySQL

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the application
